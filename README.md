# 📊 Smart Data Analyzer

A web-based data analysis tool built using Python and Streamlit that allows users to upload datasets, visualize data, and generate AI-powered insights.

---

## ✨ Features

- 📂 Upload CSV datasets
- 📄 Preview and explore data
- 📊 Interactive visualizations:
  - Bar Chart
  - Line Chart
  - Pie Chart
- 📈 Automatic statistical summary
- 🤖 AI-powered insights using Google Gemini

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- Google Gemini API
